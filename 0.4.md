sequenceDiagram
    participant User as User
    participant Browser as Browser
    participant Server as Server

    Note over User,Browser: User writes something in the text field and clicks the Save button
    User->>Browser: Enter content and click Save
    Browser->>Server: POST /data.json with new note data
    activate Server
    Server->>Server: Process the POST request and save the new note
    Server-->>Browser: Send response indicating successful save
    deactivate Server
    Browser->>Browser: Update the page display with the new note
    Note over Browser: Browser fetches the updated list of notes from the server
    Browser->>Server: GET /data.json for updated notes
    activate Server
    Server->>Server: Retrieve and prepare the updated list of notes
    Server-->>Browser: Send updated list of notes
    deactivate Server
    Browser->>User: Render and display the updated list of notes
